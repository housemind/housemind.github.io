# Benchmark

The benchmark consists of a single **data_test** package.

## Download and setup

1. **Download** the benchmark archive: [data_test.zip](https://cloud.tsinghua.edu.cn/f/7cbe9e7039f843e78236/) (Tsinghua Cloud).
2. **Extract** the archive into the HouseMind repo root so that the folder **data_test/** appears next to `run_all_tasks.py`.
3. The resulting layout should be:
   - `data_test/outline_256/` (PNG images)
   - `data_test/gt_256/`
   - `data_test/before_edit_256/`
   - `data_test/after_edit_256/`
   - `data_test/generation.jsonl`
   - `data_test/editing.jsonl`
   - `data_test/understanding.jsonl`
