# HouseMind Project Page

This repository hosts the standalone GitHub Pages site for HouseMind.

## Local Preview

Open `index.html` directly in a browser, or serve this folder with any static server.

## Deployment

- Recommended repo name: `housemind_page`
- Default page URL: `https://housemind.github.io/`
- If using a custom domain, configure it in GitHub Pages settings.
